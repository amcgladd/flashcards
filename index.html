Coding Flashcards

Choose a term to see its flashcard

Function

a set of statements that performs a task or calculates a value

Boolean

a binary variable, having two possible values called “true” and “false.”

Chaining Method

syntax for invoking multiple method calls in object-oriented programming languages

Javascript

an object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

a symbol that is used to perform an operation. Most often these operations are arithmetic (addition, subtraction, etc), but not always.

Variables

containers for storing data values

Variable Naming Conventions

best practices for naming variables

Methods

actions that can be performed on objects

Argument

JavaScript functions have a built-in object called the arguments object. The argument object contains an array of the arguments used when the function was called (invoked)

Parameters

the names listed in the function definition

Return

The return statement ends function execution and specifies a value to be returned to the function caller.

Strings

strings are used for storing and manipulating text

Undefined

property represents the primitive value undefined. It is one of JavaScript's primitive types.

NaN

(Not-a-Number) is a weirdo Global Object in javascript frequently returned when some mathematical operation failed

Escape

A character preceded by a backslash (\) is an escape sequence and has a special meaning to the compiler. The newline character (\n) has been used frequently in this tutorial in System.out.println() statements to advance to the next line after the string is printed.

Alert

This function will display text in a dialog box that pops up on the screen.

Comments

Single-line comments are used to comment a part or a full line of code in JavaScript. You can use it for either – explaining the code or debugging (commenting out to prevent execution of the browser). JavaScript syntax for single-line comment is fairly simple. All you have to do is to up two forward slashes: “//”.

jQuery

JQuery is a fast and concise JavaScript Library that simplifies HTML document traversing, event handling, animating, and Ajax interactions for rapid web development.

Attributes

The CSS attribute selector matches elements based on the presence or value of a given attribute.
